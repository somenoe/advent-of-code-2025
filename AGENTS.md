## How to run

### C#:

```bash
dotnet run 01/main.cs    # run with example input
dotnet run 01/main.cs i  # run with actual input
```

### TS:

```bash
cd 05    # change to the day's directory
bun .    # run with example input
bun . i  # run with actual input
```

### C++:

```bash
./build 10  # build the executable
./run e 10  # run with example input
./run i 10  # run with actual input
```

Or build and run in one command:

```bash
./dev e 10
```

### Rust:

```bash
rust-script 11/main.rs
```
